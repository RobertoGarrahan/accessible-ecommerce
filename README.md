# Accessible E-commerce

This project is an e-commerce model designed with a strong focus on **accessibility for visually impaired users**. It was developed as part of a university assignment, aiming to explore inclusive digital design and user experience.

The application emphasizes clear navigation, semantic structure, and accessible visual contrast, following best practices for web accessibility.

---

## â™¿ Accessibility Focus

The project was built with the following accessibility considerations:

- Clear and structured layouts
- Readable typography and high-contrast UI
- Semantic HTML to improve screen reader compatibility
- Keyboard-friendly navigation
- **Text-to-speech interaction:** when a user clicks on buttons or interactive elements, the system uses the device's speaker to verbally announce what was clicked, reinforcing feedback for visually impaired users
